## 📖 Project Overview

This project is an Open Source Software (OSS) audit of Python.
It includes both theoretical analysis and practical shell scripting tasks.

The goal is to understand:

* Open source philosophy
* Licensing (PSF License)
* Linux environment interaction
* Real-world automation using shell scripts

---

## 📂 Scripts Description

### 🔹 Script 1 — System Identity Report

Displays system information:

* Linux distribution
* Kernel version
* Current user
* Uptime and date

---

### 🔹 Script 2 — FOSS Package Inspector

* Checks if Python is installed
* Displays Python version
* Shows description of Python

---

### 🔹 Script 3 — Directory Audit Report

* Analyzes important system directories
* Displays permissions and sizes
* Finds Python configuration directory

---

### 🔹 Script 4 — Log File Analyzer

* Searches logs for keyword (default: error)
* Counts occurrences
* Displays last matching lines

---

### 🔹 Script 5 — Open Source Manifesto Generator

* Takes user input
* Generates a personal manifesto
* Saves output to a text file

---

## ▶️ How to Run the Scripts

### Step 1: Give permission

```bash
chmod +x script1.sh script2.sh script3.sh script4.sh script5.sh
```

### Step 2: Run scripts

```bash
./script1.sh
./script2.sh
./script3.sh
sudo ./script4.sh /var/log/syslog
./script5.sh
```
